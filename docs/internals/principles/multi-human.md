# Multi-human: attributed, never verified

The trust substrate for a research group (`docs/design/multi-human.md`, settled
2026-07-07): today's IMPLICIT single-actor identity made EXPLICIT as an opaque,
**harness-asserted** `actor` slug core COMPARES (`!=` / set-membership) and NEVER
verifies — no accounts, no PKI, no OS-user probe, no role vocabulary. Every
mechanism is byte-identical to today under zero/one declared actor (the D7
posture); the comparisons only mean something in a group. These rows hold the
boundary its enforcement-row table names.
