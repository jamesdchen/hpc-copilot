- Consumption: wired INSIDE `_contradiction.py::_gather_violations` (line ~272), outside the run_ids/audit_ids blocks so it fires with zero mentions. Not re-exported from the package `__init__.py` (docstring still lists five audits). `build_hook_output` (`relay_audit_stop/__init__.py`) routes violations to `_rejector_output` (block once) or, under the env-declared `stop-hook-append` capability, `_completer_output` (append `systemMessage`, bounce only on a poisoned decision); `stop_hook_active` gives block-once loop safety; everything fail-open.
